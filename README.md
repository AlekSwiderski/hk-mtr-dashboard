# Hong Kong MTR Analytics Dashboard

An interactive dashboard exploring Hong Kong's Mass Transit Railway (MTR) system through data visualization.

## Features

- **Network Overview**: Visualize all 100+ MTR stations across 10 lines
- **Fare Calculator**: Calculate fares between any two stations
- **Ridership Trends**: Historical public transport data from 2014-2024
- **Accessibility Info**: Barrier-free facilities across the network

## Data Sources

- [DATA.GOV.HK](https://data.gov.hk) - Hong Kong Open Data Portal
- MTR Open Data Platform
- Transport Department Monthly Traffic and Transport Digest

## Tech Stack

- **Python**: Data processing and analysis
- **Streamlit**: Interactive web dashboard
- **Plotly**: Data visualizations
- **Pandas**: Data manipulation

## Local Development

```bash
pip install -r requirements.txt
streamlit run app.py
```
